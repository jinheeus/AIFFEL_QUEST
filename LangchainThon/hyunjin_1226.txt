현진 lab
 
1. 폴더 설명 : C:\Users\소현진\projects\coverletter-coach-rag 에 langsmith 추가
2. 폴더명 : coverletter-coach-ragSmith


1. 프로젝트 폴더로 이동

PS C:\Users\소현진> cd C:\Users\소현진\projects\coverletter-coach-ragsmith

2. 가상환경 만들기 (한 번만 하면 됨, 이미 만들었으면 이 단계는 건너뛰어도 됨)

python -m venv venv

3. 가상환경 활성화

.\venv\Scripts\Activate.ps1

	프롬프트가 (venv) 로 시작하면 성공한 거야.

4. pip 최신화 (선택이지만 해두면 좋음)

python -m pip install -U pip

5. 필요한 라이브러리 설치
requirements.txt에 langchain, langsmith, streamlit 등이 다 들어있다고 가정하고:

python -m pip install -r requirements.txt

이미 했던
python -m pip install streamlit
은 requirements에 들어 있다면 안 해도 됨. 해도 상관은 없고.

6. .env 확인
프로젝트 폴더에 .env 파일이 있고, 그 안에 아래 같은 줄이 있는지 확인:

OPENAI_API_KEY="..."          # 원래 쓰던 키
LANGCHAIN_ENDPOINT="https://api.smith.langchain.com"
LANGCHAIN_API_KEY="lsmith_로_시작하는_키"
LANGCHAIN_TRACING_V2="true"
LANGCHAIN_PROJECT="jd-coach-demo"


7. LangSmith 평가 스크립트 실행
(내가 전에 알려준 내용대로 eval_langsmith.py 만들어 놨다는 전제)

python eval_langsmith.py


8. 웹 앱도 같이 보고 싶으면 (선택)

python -m streamlit run app.py

이거 한 번 돌아가면 터미널에 실험 이름이 찍힐 거고,
LangSmith 웹에서 Experiment로 확인할 수 있어.
